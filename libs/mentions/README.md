# mentions

## Назначение
Библиотека механики `@mentions` для текстового ввода, подсказок и выбора сущностей.

## Область ответственности
- Директива mentions и утилиты вычисления позиции каретки.
- Компоненты списка подсказок и элементов mention.
- Конфигурация mention-потока и вспомогательные функции.

## Что улучшить
- Расширить тесты курсора на multiline и нестандартные раскладки.
- Улучшить поддержку IME и мобильных клавиатур.
- Усилить accessibility dropdown подсказок.

## Техдолг
- `P0`: закрыть edge-case баги caret-позиционирования и навигации.
- `P1`: обеспечить кросс-браузерную стабильность behavior-тестами.
- `P2`: формализовать API конфигурации mentions для расширяемости.
