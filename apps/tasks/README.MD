# tasks

## Назначение
Микрофронтенд коллекции задач с несколькими представлениями: список, дерево и timeline.

## Область ответственности
- Отрисовка и переключение режимов просмотра задач.
- Фильтрация, сортировка и навигация по задачам.
- Интеграция с задачными доменными сервисами.
- Подключение в shell как remote entry.

## Что улучшить
- Привести list/tree/timeline к единому data-source и общим контрактам фильтрации.
- Добавить производительные стратегии для больших наборов данных.
- Расширить e2e-покрытие массовых операций и сложных фильтров.

## Техдолг
- `P0`: устранить расхождения бизнес-логики между представлениями задач.
- `P1`: ввести централизованное состояние для синхронизации list/tree/timeline.
- `P2`: стандартизировать кэширование и инвалидацию фильтров.
